# Алгоритмы сортировки

- [Быстрая сортировка](sorts/quick.go) (O(n log n) в среднем / O(n^2) в худшем)
- [Сортировка слиянием](sorts/merge.go) (O(n log n))
- [Пирамидальная сортировка](sorts/heap.go) (O(n log n))
- [Timsort](sorts/tim.go) (O(n log n))
- [Поразрядная сортировка](sorts/radix.go) (O(n))
- [Сортировка подсчётом](sorts/counting.go) (O(n + k))
- [Блочная сортировка](sorts/bucket.go) (O(n + k))
- [Сортировка вставками](sorts/insertion.go) (O(n^2))

## Структура проекта

- `main.go` — точка входа, которая просто делегирует запуск пакету `internal/app`.
- `internal/app/interactive.go` — логика текстового интерфейса и работа с пользовательским вводом.
- `internal/app/visualization.go` и `internal/app/visual_sorts.go` — движок визуализатора и пошаговые реализации всех алгоритмов.
- `sorts/` — чистые реализации алгоритмов, которые можно переиспользовать в других программах.

## Визуальный режим

Приложение умеет запускать все алгоритмы параллельно и выводить в консоль «столбцы», чтобы можно было наблюдать процесс сортировки.

1. Скомпилируйте или запустите проект: `go run .`
2. В главном меню выберите пункт `2 - Визуализировать работу всех алгоритмов одновременно`.
3. На экране появится набор блоков из символов `▁ ... █`, где каждый блок — это копия исходных данных для конкретного алгоритма.
4. Алгоритмы работают медленно специально (добавлена задержка между шагами), поэтому можно наблюдать за тем, как столбцы постепенно выстраиваются от меньших к большим.
