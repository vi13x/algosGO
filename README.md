# Алгоритмы сортировки

В репозитории собраны наглядные реализации классических алгоритмов сортировки на Go. Каждая реализация поддерживает передачу колбэка для визуализации шагов и используется из интерактивного `main.go`.

## Список алгоритмов
- [Быстрая сортировка](sorts/quick.go) — `O(n log n)`
- [Сортировка слиянием](sorts/merge.go) — `O(n log n)`
- [Пирамидальная сортировка](sorts/heap.go) — `O(n log n)`
- [Timsort](sorts/tim.go) — `O(n log n)`
- [Поразрядная сортировка](sorts/radix.go) — `O(n)`
- [Сортировка подсчётом](sorts/counting.go) — `O(n + k)`
- [Блочная сортировка](sorts/bucket.go) — `O(n + k)`
- [Сортировка вставками](sorts/insertion.go) — `O(n^2)`

## Как запустить визуализацию
```bash
go run .
```

Приложение предложит выбрать конкретный алгоритм или запустить одновременную визуализацию всех сортировок. Для замедления выполнения применяется пауза между шагами, поэтому лучше запускать из отдельного терминала.
